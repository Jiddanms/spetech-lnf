
# 🔍 Spetech Lost and Found (LNF)
**Sistem Informasi Pelaporan Barang Hilang Berbasis Web dengan Integrasi QR Lokasi**

Proyek Karya Tulis Ilmiah (KTI) untuk **SMP Techno**.

---

## 🚀 Inovasi Utama
Web ini mengintegrasikan **QR Code Lokasi Pelaporan**. Penemu cukup memindai QR Code di lokasi penemuan (seperti Kantin atau Lab), dan form pelaporan akan otomatis terisi lokasinya secara presisi.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Dark Theme), Vanilla JavaScript.
- **Backend:** Cloudflare Pages Functions (Serverless).
- **Database:** Cloudflare D1 (SQL-based).
- **Hosting:** Cloudflare Pages.

## 📁 Struktur Folder
- `assets/`: File statis (CSS, JS, Gambar).
- `functions/`: Logika backend (API).
- `index.html`: Entry point utama aplikasi.

## ⚙️ Cara Menjalankan Lokal
1. Clone repository ini.
2. Jalankan `npm install`.
3. Jalankan `npm run dev` untuk memulai server lokal.
